# 📊 Mini-CRM System (Java)

Eine leichtgewichtige, konsolenbasierte Kundenverwaltung, entwickelt in Java. Dieses Tool hilft kleinen Betrieben dabei, ihre Kundendaten strukturiert zu erfassen und Fehler bei der Eingabe zu vermeiden.

## 💡 Das Problem, das dieses Tool löst
Viele kleine Unternehmen nutzen noch immer Zettelwirtschaft oder unübersichtliche Dokumente für ihre Kunden. Dieses Programm bietet eine fehlerresistente Eingabemaske und bündelt alle Daten zentral.

## ✨ Kernfunktionen
* **Sichere Datenerfassung:** Intelligente Fehlerabfangung (z.B. Blockierung von Buchstaben bei Kundennummern) verhindert Abstürze durch falsche Nutzereingaben.
* **Dynamische Speicherung:** Verwaltung einer beliebigen Anzahl von Kunden zur Laufzeit.
* **Automatisierter Excel-Export:** Generiert auf Knopfdruck eine saubere `.csv`-Datei, die direkt für die Buchhaltung oder in Excel weiterverwendet werden kann.

## 🛠️ Tech Stack
* **Sprache:** Java 
* **Konzepte:** Objektorientierte Programmierung (OOP), Datenstrukturen (ArrayList), Exception Handling (try-catch), File I/O (Daten-Export).
